    这个项目是对我平时开发时, 对异常处理的总结, 是在MVC开发模式前提下, 在众多项目的开发过程中总结出来的, **针对MVC项目的**异常处理规范

- design-pattern-java  
    该项目是学习实际模式的代码记录, 里面有具体的README
